# Suggested Commands

From the repo root:

```bash
cargo check
cargo build --release
cargo test
cargo fmt
cargo clippy
```

For the extension:

```bash
cd extension
npm install
npm run build
npm run dev
npm run typecheck
```

Useful project commands:

```bash
autocli --help
autocli doctor
autocli completion zsh
```

System utilities that are commonly useful on macOS:

```bash
git status
ls -la
find .
rg "pattern"
```
